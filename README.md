# GreenEden - Calculadora de Impacto Ambiental de Transações

## 📖 Sobre o Projeto

O **EcoImpact** é uma ferramenta desenvolvida para comparar o impacto ambiental entre **transações realizadas com cartões físicos e transações digitais**.

O projeto foi criado a partir de um desafio proposto pela Edenred, que busca demonstrar os benefícios ambientais da digitalização de seus serviços.

A aplicação tem como objetivo fornecer **dados claros e mensuráveis** que evidenciem como a migração para soluções digitais pode contribuir para a **redução de impactos ambientais**, como emissões de carbono, uso de plástico e consumo de papel.

## 🎯 Objetivo

Desenvolver uma **calculadora de impacto ambiental** capaz de:

- Comparar **transações físicas e digitais**
- Estimar **redução de emissões de CO₂**
- Calcular **economia de plástico e papel**
- Demonstrar **benefícios ambientais da digitalização**

---

## 🚨 Problema:

Atualmente, muitas empresas ainda utilizam **cartões físicos para distribuição de benefícios**, o que gera impactos ambientais significativos.

Entre os principais problemas estão:

- Uso de **plástico na produção de cartões**
- Consumo de **papel e embalagens**
- **Logística de transporte** para distribuição
- **Emissões de carbono** associadas ao processo
- **Resíduos plásticos e eletrônicos** após descarte

Além disso, existe **resistência de clientes à migração para soluções digitais**, principalmente pela falta de dados claros que demonstrem os benefícios ambientais dessa mudança.

---

## 💡 Solução Proposta

O projeto propõe o desenvolvimento de uma **calculadora digital de impacto ambiental**, capaz de apresentar comparações claras entre transações físicas e digitais.

A ferramenta permitirá visualizar indicadores como:

- 🌍 Redução de **emissões de CO₂**
- ♻️ Economia de **plástico**
- 📄 Redução no consumo de **papel**
- 🌳 Equivalência em **impacto ambiental positivo**

Essas informações podem ajudar empresas e usuários a **compreender melhor os benefícios da digitalização**.

---

## ⚙️ Funcionalidades

- Comparação entre **transação física e digital**
- Estimativa de **impacto ambiental**
- Visualização clara dos resultados
- Apresentação de **indicadores ambientais**
